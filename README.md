# Crunch Together

Crunch Together is a browser extension for both Google Chrome and Firefox to synchronize Crunchyroll videos across multiple computers. I made this because [Roll Together](https://github.com/samuraiexx/roll_together) got discontinued.

This tool is currently still under development and super buggy, so as of right now you can only really install it from source, but this will hopefully be better over the next days/weeks.
